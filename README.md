# 🌞 Adani Green Solar Energy - SQL Project

## 📌 Project Overview
Analysis of solar energy plant data using MySQL.
This project covers plant performance, energy output trends,
and maintenance impact analysis.

## 🛠️ Tools Used
- MySQL 8.0
- MySQL Workbench

## 🗃️ Database Structure
- **solar_plants** – Plant information & location
- **energy_output** – Daily energy generation data
- **maintenance_logs** – Maintenance records

## 📊 Key Analysis Performed
- Energy output trends across plants
- Plant performance comparison
- Maintenance impact on energy output
